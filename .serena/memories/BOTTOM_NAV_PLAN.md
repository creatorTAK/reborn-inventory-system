# ボトムナビ導入計画

**作成日**: 2025-12-14
**最終更新**: 2025-12-14
**ステータス**: 検討中（ユーザーが詳細をまとめ中）

## 現状の課題
- トップメニューのミニヘッダー（チャット、タスク、マイページ）が浮いている感じ
- サブメニュー画面が2段ヘッダー（FURIRAヘッダー + サブヘッダー）で領域を圧迫

---

## 決定事項

### ボトムナビ構成（5つ）

```
┌─────────────────────────────────────────┐
│                                         │
│  🏠     💬    [ 📷 ]    ✅      ❓     │
│ ホーム チャット カメラ  タスク  ヘルプ   │
│                                         │
└─────────────────────────────────────────┘
```

| 位置 | アイコン | 機能 | 役割 |
|-----|---------|------|------|
| 左端 | 🏠 | ホーム | ナビゲーションの起点 |
| 左寄り | 💬 | チャット | チーム連携 |
| **中央** | **📷** | **カメラ** | **メイン操作（商品撮影）- 大きめボタン** |
| 右寄り | ✅ | タスク | やることリスト |
| 右端 | ❓ | ヘルプ | 困った時のサポート（FAQ/操作ガイド） |

### トップメニュー画面
```
┌─────────────────────────────────┐
│                           👤   │  ← マイページのみ右上
│         FURIRA ロゴ             │
│    [メニューグリッド]            │
├─────────────────────────────────┤
│  🏠   💬  [📷]  ✅   ❓        │  ← ボトムナビ（5つ）
└─────────────────────────────────┘
```

### サブメニュー画面
```
┌─────────────────────────────────┐
│  ←  📝 商品登録       👤   ≡  │  ← デニムブルー背景（1段に統一）
├─────────────────────────────────┤
│      [コンテンツ]               │
├─────────────────────────────────┤
│  🏠   💬  [📷]  ✅   ❓        │  ← 同じボトムナビ
└─────────────────────────────────┘
```

---

## 変更ポイント
1. FURIRAヘッダー行をサブメニューから削除（ホームボタンで代替）
2. サブヘッダーをデニムブルー背景に変更
3. ハンバーガーメニューは右端に残す（アクセス性確保）
4. チャット・タスクはボトムナビへ移動
5. マイページアイコンは右上に残す
6. ボトムナビはindex.html（親フレーム）に配置 → 全画面で常時表示
7. センターにカメラボタン（大きめ・浮き出しデザイン）
8. 右端にヘルプ（FAQ/操作ガイド）

---

## メリット
- ヘッダーが1段になりスッキリ
- デニムブルーのテーマ感が増す
- メニューの操作域が広がる
- カメラが最もアクセスしやすい位置（商品登録のメイン操作）
- ヘルプで初心者ユーザーも安心

## デザインガイドライン（メルカリ・LINE・Amazon参考）

大手アプリの共通パターンに合わせる:

| 要素 | 仕様 |
|-----|------|
| **背景** | 白（#FFFFFF） |
| **上部ボーダー** | 薄いグレー線（1px） |
| **アイコン** | 線画（アウトライン）スタイル - Bootstrap Icons等 |
| **非選択時** | グレー（#9CA3AF） |
| **選択時** | デニムブルー（#40B4E5） |
| **ラベル** | アイコン下に小さく表示（10-11px） |
| **センターカメラ** | 少し大きめ or 丸で強調 |

```
┌─────────────────────────────────────────┐
│  ←  📝 商品登録       👤   ≡          │  ← デニムブルーヘッダー
├─────────────────────────────────────────┤
│                                         │
│            [コンテンツ]                  │
│                                         │
├─────────────────────────────────────────┤  ← 薄いボーダー(#E5E7EB)
│  🏠     💬    [📷]    ✅      ❓       │  ← 白背景
```
