# オークファンAPI連携計画

**作成日**: 2025-12-12
**重要度**: 🔴 最高（物販の要）

---

## 概要

オークファンは700億件以上の売買データを保有し、**法人向けにAPIを提供**している。
2024年5月に**メルカリと公式データ連携**を発表しており、規約違反なしで価格調査が可能。

---

## オークファンの強み

- **700億件以上**の売買データ
- **メルカリ公式データ連携**（2024年5月〜）
- Yahoo!オークション、楽天市場、Amazon、1688.com対応
- **過去10年分**の相場データ
- **法人向けAPI提供**

---

## REBORN連携で実現できること

### 1. 商品登録時の自動価格提案

```
商品登録（ブランド、アイテム、状態、サイズ入力）
    ↓
オークファンAPI呼び出し
    ↓
【取得データ】
  - メルカリ過去取引価格（平均・最高・最低）
  - ヤフオク落札価格
  - 価格推移トレンド（上昇/下降/安定）
    ↓
AI分析（Gemini）
    ↓
【出力】
  - 推奨販売価格: ¥4,500
  - 価格レンジ: ¥3,800〜¥5,200
  - 売れるまでの予測日数: 3〜7日
  - 「この価格帯は出品数が少なく狙い目です」
```

### 2. 在庫の価値評価・アラート

| 機能 | 内容 |
|------|------|
| **在庫価値の可視化** | 全在庫の現在の市場価値を自動計算 |
| **値下げアラート** | 「AA-1001の相場が10%下落。値下げ推奨」 |
| **売り時通知** | 「このブランドの相場が上昇中。今が売り時」 |
| **不良在庫検知** | 「30日以上滞留 + 相場下落 → 早期処分推奨」 |

### 3. 仕入れ判断支援

```
仕入れ検討時
    ↓
商品情報入力（ブランド、アイテム、状態）
    ↓
オークファンAPI
    ↓
【判定結果】
  - 市場相場: ¥5,000
  - 仕入れ価格: ¥2,000
  - 想定利益: ¥2,500（手数料・送料込み）
  - 利益率: 50%
  - 判定: ✅ 仕入れ推奨
```

### 4. プラットフォーム別価格最適化

```
【同一商品の価格差分析】
  - メルカリ平均: ¥4,200
  - ヤフオク平均: ¥4,800（+14%）
  - メルカリShops: ¥5,100（+21%）
    ↓
【提案】
  - メルカリ: ¥4,000（早く売りたい場合）
  - ヤフオク: ¥4,500（やや高めで出品可）
  - メルカリShops: ¥4,800（ショップ信頼度で上乗せ可）
```

### 5. レポート・分析ダッシュボード

| レポート | 内容 |
|----------|------|
| **月次収益予測** | 在庫の市場価値から売上予測 |
| **トレンド分析** | 「このブランドは3ヶ月で相場20%上昇」 |
| **利益率ランキング** | 高利益商品のパターン分析 |
| **仕入れ推奨レポート** | 「今狙い目のブランド・アイテム」 |

### 6. SaaS機能としての差別化

```
【フリープラン】
  - 価格調査なし（手動でオークファン確認）

【スタンダードプラン】
  - 商品登録時に相場表示
  - 月100件まで価格調査

【プロプラン】 ← 法人向け
  - 無制限価格調査
  - 自動値下げ提案
  - 仕入れ判断AI
  - 在庫価値レポート
```

## 技術的な実装イメージ

```
n8nワークフロー:

Firestore Trigger（商品登録）
    ↓
オークファンAPI（価格データ取得）
    ↓
Gemini API（価格分析・提案文生成）
    ↓
Firestoreに推奨価格・分析結果保存
    ↓
FCM通知「AA-1001の推奨価格: ¥4,500」
```

---

## なぜこれが「物販の要」なのか

1. **仕入れ判断**: 利益が出るか出ないかを事前に判断
2. **価格設定**: 相場に基づいた最適価格で早期販売
3. **在庫管理**: 滞留リスクを可視化、値下げタイミングを自動提案
4. **プラットフォーム戦略**: どこで売れば最も利益が出るか

→ **物販の全フェーズでデータドリブンな意思決定が可能に**
