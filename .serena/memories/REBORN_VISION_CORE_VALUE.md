# REBORNの本質的価値とビジョン

**作成日**: 2025-12-12
**ステータス**: コアビジョン（最重要）

---

## 問題提起：従来の物販業界

### 属人的・非効率な現状
- **仕入れ**: ベテランの「目利き」に依存
- **撮影**: 個人のスキル差が大きい
- **出品**: 1商品15分、手作業でコピペ
- **価格設定**: 勘で決める、相場調査は手動
- **在庫管理**: Excelか頭の中、把握しきれない
- **発送**: 誰がどこまでやったか分からない
- **分析**: やる時間がない、データがない

### 結果
- スケールできない（人が増えると混乱）
- 属人化（その人がいないと回らない）
- 利益率が安定しない
- 疲弊する

---

## REBORNが実現する世界

### データ駆動・チーム連携
- **仕入れ**: データ根拠で**誰でも**判断可能
- **撮影**: 推奨ツール案内、基準明確化
- **出品**: AI自動生成、**3分で完了**
- **価格設定**: オークファン連動、最適価格自動提案
- **在庫管理**: リアルタイム共有、どこからでも確認
- **発送**: チェックリスト連携、進捗可視化
- **分析**: 自動蓄積、傾向が見える

### 結果
- **スケール可能**（人を増やせば売上増）
- **標準化**（誰がやっても同じ品質）
- **利益率向上**（データで判断）
- **持続可能**

## 2つのデータソースの掛け合わせ

### オークファン（外部市場データ）
- 700億件の過去取引データ
- メルカリ/ヤフオク/楽天の相場
- トレンド（価格上昇/下落）

### REBORN（自社販売データ）
- 実際に売れた価格・日数
- ブランド×カテゴリ×状態別の実績
- 仕入れ価格 vs 販売価格（実利益）
- どの仕入れ先が良かったか

### 掛け合わせで実現する機能
- **仕入れ前査定**: 写真撮影 → AI判定 → 推奨仕入れ価格・予想利益
- **ブランド別利益率**: 自社実績からの分析
- **仕入れ先評価**: どこで仕入れると利益率高いか
- **トレンドアラート**: 相場上昇中ブランドの通知
- **死筋予測**: 回転率低いカテゴリの警告

## FURIRAの社会的ミッション

### 背景：日本の社会課題
- **物価高による生活困窮者の増加**
- 働きたくても働けない人が多数存在

### ターゲット層

| 層 | 課題 |
|---|------|
| 子育て中のママ | 外で働けない、時間が不規則 |
| シルバー世代 | 年金だけでは不足、体力的に外勤困難 |
| 介護離職者 | 家を離れられない |
| 障がいのある方 | 通勤・職場環境のハードル |
| 地方在住者 | 仕事自体が少ない |

### FURIRAが解決すること

**従来の副業の壁:**
- スキルが必要
- 時間拘束がある
- 初期投資が必要
- リスクが高い
- 判断力が必要

**FURIRAでの作業:**
- スキル不要（システムがガイド）
- 時間自由（いつでもどこでも）
- 初期投資なし（仕入れは経営者が担当）
- リスクなし（作業した分だけ報酬）
- 判断不要（データとチェックリストがある）

### 作業者の働き方

| 役割 | 作業内容 | 必要なもの | 時間 |
|------|---------|-----------|------|
| 撮影担当 | 商品を撮影してアップ | スマホ | 1商品5分 |
| 出品担当 | AI生成文を確認・出品 | スマホ/PC | 1商品3分 |
| 梱包担当 | チェックリスト通りに梱包 | 梱包材 | 1商品10分 |
| 発送担当 | 集荷依頼 or コンビニ持込 | - | まとめて処理 |

### 社会インフラとしてのFURIRA

> **「自宅にいながら、誰でも、いつでも、副収入を得られるインフラ」**

1. **誰でも参加できる** - スキル・年齢・場所を問わない
2. **必要な時に働ける** - 強制ではなく、自分のペースで
3. **持続可能** - ビジネスが回れば仕事が生まれ続ける
4. **スケール可能** - 全国どこでも、何人でも
