# セッション開始時の初期化

新しいセッションを開始します。プロジェクトの最新状態を把握するため、以下のドキュメントを読み込んでください：

## 必須ドキュメント（常に読み込み）

1. **REBORN_INVENTORY_CLAUDE.md** - プロジェクト全体の概要、技術スタック、開発ルール、最新更新
2. **docs/issues.md** - 現在の未完了Issue（優先的に対応すべき課題）

## 追加ドキュメント（必要に応じて）

タスクに応じて、以下のドキュメントを追加で読み込んでください：

- **Issue起票が必要な場合**: `docs/TDD_POLICY.md`
- **PWA/FCM関連の作業**: `REBORN_INVENTORY_PWA_SETUP.md`
- **開発環境・ツール確認**: `REBORN_INVENTORY_SERVICES.md`
- **API安全ルール**: `API_SAFETY_CRITICAL.md`（新しいAPI導入時）

---

**トークン効率化のため、必須ドキュメントのみを読み込んでいます（約5,200トークン）**
