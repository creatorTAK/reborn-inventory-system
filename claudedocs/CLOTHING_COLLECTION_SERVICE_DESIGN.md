## 1. サービス概要

### コンセプト
「着なくなったファッションアイテムを無料で回収」- 処分の手間をゼロにするサービス

### URL
`furira.jp/collect`

### 回収対象
ファッション関連全般:
- 衣類（トップス、ボトムス、アウター等）
- 靴
- バッグ
- アクセサリー
- 帽子、マフラー等の小物

### ビジネスモデル
```
一般消費者（不要な服）
    │
    │ 無料回収
    ↓
FURIRA（仕入れ）
    │
    │ 仕分け
    ├─ 良品 → メルカリ等で販売（利益）
    ├─ 難あり → 安価で販売（薄利）
    └─ 使用不可 → 無料引取業者（コストゼロ）
```

### 対象エリア（Phase 1）
- 岡山市内限定

### ターゲット
- 服を処分したいが面倒な人
- メルカリ出品は手間という人
- 捨てるのはもったいないと感じる人

## 4. Firestoreデータ設計

### 4.1 コレクション: pickup_requests

```javascript
{
  id: "auto-generated",

  // 申込者情報
  name: "山田太郎",
  phone: "090-1234-5678",
  postalCode: "700-0001",
  address: "岡山市北区○○1-2-3",

  // 希望
  preferredDay: "weekend",      // weekday, weekend, any
  preferredTime: "afternoon",   // morning, afternoon, evening, any
  estimatedAmount: "2-3",       // 1, 2-3, 4+
  note: "駐車場あります",

  // ステータス管理
  status: "new",                // new, contacting, scheduled, collected, processed
  scheduledDate: null,          // 確定した回収日
  scheduledTime: null,          // 確定した回収時間

  // メタ情報
  createdAt: Timestamp,
  updatedAt: Timestamp,
  collectedAt: null,            // 回収完了日時
  processedAt: null,            // 仕分け完了日時

  // 回収結果（回収後に記録）
  result: {
    totalWeight: null,          // 総重量(kg)
    goodItems: null,            // 良品点数
    fairItems: null,            // 難あり点数
    unusableItems: null         // 使用不可点数
  }
}
```

### 4.2 ステータス遷移

```
new（新規申込）
  ↓ 電話連絡
contacting（連絡中）
  ↓ 日程確定
scheduled（回収予定）
  ↓ 回収完了
collected（回収済）
  ↓ 仕分け完了
processed（処理完了）
```

## 9. FAQ案

**Q. どんなものが回収対象ですか？**
A. ファッション関連全般です。衣類、靴、バッグ、アクセサリー、帽子など何でもOKです。

**Q. ブランド品じゃなくても大丈夫？**
A. はい、ブランド問わず何でも回収します。

**Q. 汚れや破れがあっても大丈夫？**
A. 大丈夫です。状態に関わらずお引き取りします。

**Q. 本当に無料ですか？**
A. はい、完全無料です。費用は一切かかりません。

**Q. 回収日はいつですか？**
A. お申込み後、お電話またはLINEで日程を調整させていただきます。

**Q. どうやって連絡が来ますか？**
A. お申込み時に選択いただいた方法（電話またはLINE）でご連絡します。
