## 🎯 カラー使用ガイドライン

### Primary（スカイブルー）の使い方

```
✅ 推奨用途
- ヘッダー背景（グラデーション）
- ナビゲーションバー
- メッセージバブル（送信済み）
- 進捗バー
- リンクテキスト（Dark使用）
- ボタン（標準操作）

❌ 非推奨用途
- 本文テキスト（コントラスト不足）
- 小さいアイコン（視認性低下）
- エラーメッセージ（Dangerカラー使用）
```

### Accent（オレンジ）の使い方

```
✅ 推奨用途
- CTAボタン（重要操作）
- 通知バッジ
- 強調テキスト
- アクティブ状態
- 数値・統計のハイライト

❌ 非推奨用途
- 大面積の背景（目に優しくない）
- エラー表示（Dangerと混同）
- ヘッダー背景（Primaryが適切）
```
