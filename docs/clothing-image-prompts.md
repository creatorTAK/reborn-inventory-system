# 衣類イラスト Canva AI用プロンプト集

傷・汚れマーキング機能用の衣類イラスト作成プロンプトです。
1つずつCanva AIに入力してください。

---

## 共通スタイル指示（最初に読んでおく）

すべての画像で以下のスタイルを統一してください：
- 白背景
- 黒の輪郭線のみ（塗りつぶしなし）
- 線の太さは均一
- 正面図
- 影やグラデーションなし
- 装飾なし（ボタン、ロゴ、柄なし）
- サイズ: 正方形
