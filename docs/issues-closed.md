# Issues（完了・アーカイブ）

このファイルは、REBORN Inventoryプロジェクトの**完了したIssue**をアーカイブします。

**運用ルール：**
- `docs/issues.md` で ✅ DONE になったIssueをここに移動
- 新しい完了Issueは最上部に追加（日付順）
- 削除せずに保管（将来の参考資料として活用）

- [TDD_POLICY.md](./TDD_POLICY.md) - Issue管理ルール詳細
