管理番号設定
管理番号の自動生成ルールを設定します
管理番号の設定がまだありません
下の「フォーマットを選択」から設定してください

管理番号フォーマット

棚番号 + 連番
生成される管理番号の例
AA-001
フォーマットを選択・編集
+
シンプル連番
001 → 002 → 003
シンプルに番号だけで管理。少量の商品を扱う方や、初めての方向け
棚番号 + 連番 おすすめ
AA-001, AB-001, BA-001
収納場所（棚・ラック）ごとに分類。在庫の場所がすぐわかる
カテゴリ + 日付 + 連番
T-260101-001, B-260101-002
商品カテゴリと登録日で分類。いつ・何を登録したか一目瞭然
棚番号 + 日付 + 連番
AA-260101-001, AB-260101-002
収納場所と登録日の両方で管理。大量の在庫を扱う方向け
カテゴリ + ランク + 連番
T-S-001, B-A-002, O-B-003
カテゴリと商品ランク（S/A/B/C）で分類。価値別の管理に最適
